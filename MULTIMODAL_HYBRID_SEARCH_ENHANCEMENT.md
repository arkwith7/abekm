# 멀티모달 하이브리드 검색 향상 가이드

## 개요

이 문서는 InsightBridge에서 멀티모달 데이터를 효과적으로 검색하기 위한 하이브리드 검색 시스템의 설계와 구현을 다룹니다.

## 하이브리드 검색 아키텍처

### 검색 레이어 구조
- 키워드 검색 (Elasticsearch)
- 벡터 검색 (pgvector)  
- 메타데이터 검색 (PostgreSQL)
- 결과 융합 및 재랭킹

### 지원 검색 모드
- 텍스트 검색: 키워드 + 의미 검색 하이브리드
- 이미지 검색: CLIP 기반 이미지-텍스트 매칭
- 구조화 검색: 테이블/차트 내 데이터 검색
- 크로스 모달: 텍스트로 이미지 찾기, 이미지로 관련 텍스트 찾기

이 하이브리드 검색 시스템을 통해 사용자는 다양한 형태의 쿼리로 멀티모달 콘텐츠를 효과적으로 검색할 수 있습니다.
