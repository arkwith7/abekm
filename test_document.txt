# 테스트 문서 내용

## 웅진 WKMS 시스템 개요

웅진 지식관리시스템(WKMS)은 기업의 지식 자산을 체계적으로 관리하는 플랫폼입니다.

### 주요 기능
1. 문서 업로드 및 분류
2. 벡터 기반 의미 검색
3. 한국어 자연어 처리
4. 권한 기반 접근 제어

### 기술 스택
- Backend: FastAPI + PostgreSQL + pgvector
- Frontend: React + TypeScript
- AI: AWS Bedrock + Kiwipiepy

이 시스템은 웅진의 조직구조를 반영한 계층적 권한 관리를 제공합니다.
