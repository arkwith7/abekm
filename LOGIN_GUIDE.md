## 🎨 디자인 특징

### 색상 팔레트 (WKMS 브랜드 기준)
- **Primary Blue**: `#1976d2` (기본 파란색)
- **Light Blue**: `#42a5f5` (밝은 파란색)
- **Dark Blue**: `#0d47a1` (어두운 파란색)
- **Background Gradient**: `#e3f2fd` → `#f3e5f5`

### UI 컴포넌트
- 3D 아이소메트릭 스타일 일러스트
- 그라데이션 배경과 부유하는 기하학적 도형
- 깔끔한 화이트 로그인 폼
- 반응형 디자인 (모바일 최적화)

### 인터랙션
- 비밀번호 표시/숨김 토글
- 로딩 상태 애니메이션
- 호버 효과 및 포커스 상태
- 폼 유효성 검사

## 🔧 기술적 특징

### 보안
- 클라이언트 사이드 폼 검증
- JWT 토큰 기반 인증
- 로컬 스토리지 토큰 관리
- bcrypt 패스워드 해싱 (백엔드)

### 개발 모드
- 개발 환경에서는 모든 로그인 허용
- Mock 사용자 데이터 사용
- 자동 토큰 생성 및 저장

## 🚀 실행 방법

```bash
# 프론트엔드만 개발 서버로 실행
cd frontend
npm install
npm start

# 또는 전체 스택 Docker로 실행
cd /home/admin/wkms-aws
./setup.sh
```

## 📱 반응형 지원

- **데스크톱**: 1200px+
- **태블릿**: 768px - 1199px  
- **모바일**: 320px - 767px

모든 화면 크기에서 최적화된 사용자 경험을 제공합니다.
