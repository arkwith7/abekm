# 비밀번호 관리 가이드

## 개요

이 문서는 InsightBridge 시스템에서 사용자 비밀번호 관리 및 보안 정책에 대해 설명합니다.

## 비밀번호 정책

### 최소 요구사항

- 최소 길이: 8자 이상
- 복합성: 대문자, 소문자, 숫자, 특수문자 중 3가지 이상 포함
- 금지 패턴: 연속된 문자나 숫자, 사용자 ID와 동일한 문자열 금지
- 이전 비밀번호: 최근 5개 비밀번호와 중복 불가

## 보안 정책

### 계정 잠금 정책

- 최대 로그인 시도 횟수: 5회
- 잠금 시간: 30분
- 자동 해제: 시간 만료 후 자동 해제

### 세션 관리

- 세션 만료: 30분 비활성 시
- 강제 로그아웃: 비밀번호 변경 시 모든 세션 무효화

이 가이드를 통해 안전한 비밀번호 관리 시스템을 구축할 수 있습니다.
