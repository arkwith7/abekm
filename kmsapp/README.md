# KMS App - Legacy 모듈

## 개요

이 디렉토리는 InsightBridge 시스템의 레거시 KMS(Knowledge Management System) 구성요소를 포함합니다.

## 상태

⚠️ DEPRECATED: 이 모듈은 더 이상 활발하게 개발되지 않으며, 새로운 backend/ 및 frontend/ 구조로 대체되었습니다.

## 마이그레이션 정보

### 새로운 구조로 이동된 항목들

1. 데이터 모델: backend/app/models/로 이동
2. API 엔드포인트: backend/app/api/로 이동
3. 비즈니스 로직: backend/app/services/로 이동
4. 프론트엔드 컴포넌트: frontend/src/로 이동

### 정리 계획

이 레거시 모듈은 모든 기능이 새 구조로 완전히 마이그레이션 완료된 후 제거될 예정입니다.
