# PostgreSQL 한국어 검색 확장 가이드

## 개요

이 문서는 InsightBridge 시스템에서 PostgreSQL을 사용한 효과적인 한국어 검색 기능 구현을 위한 확장 모듈 설정을 설명합니다.

## 한국어 검색의 특수성

### 한국어 텍스트의 특징

- 교착어: 어미와 조사가 단어에 붙어 형태가 변함
- 복합어: 여러 어근이 결합된 단어가 많음
- 문맥 의존성: 문맥에 따라 의미가 달라짐

## 확장 모듈 구성

### 필수 확장 모듈 설치

```sql
CREATE EXTENSION IF NOT EXISTS pg_trgm;  -- 삼중자 기반 유사도 검색
CREATE EXTENSION IF NOT EXISTS unaccent; -- 악센트 제거
CREATE EXTENSION IF NOT EXISTS vector;   -- 의미 기반 벡터 검색
```

### 한국어 검색용 인덱스 생성

```sql
-- 삼중자 기반 유사도 검색 인덱스
CREATE INDEX idx_documents_content_trgm ON documents USING gin(content gin_trgm_ops);

-- 전체 텍스트 검색 인덱스
CREATE INDEX idx_documents_fts_korean ON documents USING gin(to_tsvector('korean', content));
```

이 한국어 검색 확장을 통해 InsightBridge는 한국어 문서에 대한 정확하고 효과적인 검색 기능을 제공할 수 있습니다.
